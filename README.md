## Notes App

A simple and clean notes application made using **HTML, CSS and JavaScript**.  
You can create notes, edit them, delete them and search through them easily.  
All notes are stored in the browser using **LocalStorage**, so they donâ€™t get deleted after refresh.

## Features
- Add new notes  
- Title + content support  
- Auto date/time & character count  
- Edit and delete notes  
- Search bar for quick filtering  
- Sidebar + main editor layout  
- Fully responsive (mobile + desktop)  
- Saves everything in LocalStorage

- **CSS**
